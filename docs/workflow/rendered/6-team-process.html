    %% DESIGNER WORKS
    N2 --> D1["🎨 Designer\nopens campaign in gallery"]:::designer
    D1 --> D2["Reviews all creatives\nper persona, per version, per format"]:::designer

    D3 --> DG{Designer\nsatisfied with quality?}:::gate
    DG -->|"Needs more work"| D3
    DG -->|"Done"| D4["✅ Designer submits finals"]:::designer

    %% MARKETING FINAL APPROVAL
    D4 --> N3["📣 Teams + Outlook Notification\n'Designer finals submitted'"]:::notify
    N3 --> M4["👔 Marketing Manager\nfinal review of polished creatives"]:::marketing
    M4 --> FG{Final approval?}:::gate

    N4 --> R4["👤 Recruiter\naccesses creative library\nbuilds UTM tracked links\nshares with candidates"]:::recruiter

    %% AGENCY HANDOFF
    M5 --> M6["👔 Marketing Manager\ngenerates agency magic link"]:::marketing
    M6 --> N5["📣 Teams + Outlook Notification\n'Campaign package ready'\n+ magic link"]:::notify
    N5 --> A1["📊 Paid Media Agency\nopens magic link portal"]:::agency
    A1 --> A2["Downloads creatives\ngets targeting data\ngets ad copy + UTM links\ngets landing page URLs\nlaunches ad campaigns"]:::agency

Recruiter

Initiates + receives
  • Receives JD from hiring team
  • Fills intake form (5 steps)
  • Submits request
  • Waits for approval notification
  • Accesses creative library
  • Builds UTM tracked links
  • Shares links with candidates

Marketing Manager

Reviews + approves + delivers
  • Reviews AI-generated campaign
  • Edits media strategy inline
  • Adjusts budgets + targeting
  • Enters landing page URLs
  • Approves for designer
  • Final approval after design polish
  • Sends magic link to agency

Designer

Polishes + perfects
  • Reviews creatives in gallery
  • Quick retouch (faces, backgrounds)
  • Adjusts text overlays + styles
  • Regenerates if needed
  • Exports to Figma
  • Pushes finals back
  • Submits for approval

Paid Media Agency

Receives + launches
  • Receives magic link
  • Reviews strategy + targeting
  • Downloads all creatives
  • Gets ad copy per creative
  • Gets UTM tracked links
  • Gets landing page URLs
  • Launches campaigns in ad managers
